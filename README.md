Using the VÃ¤sttrafik API (https://developer.vasttrafik.se/portal), this React app lets users search for stops to see when the next tram or bus arrives. Uses Node.js/Express.js for fetching requests from the API. 

Live demo: https://vttimeboard.herokuapp.com/

Setup Instructions:
1. Go to https://developer.vasttrafik.se/portal and register! 
2. Type "npm install" in the terminal
3. Type "npm run webpack" in the terminal
4. Type "npm start" to start a localhost
